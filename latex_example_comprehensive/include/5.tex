\section{Expressions, equations}


This is inline math: $x=4$. 

This is displayed math: 
$$x = 4$$ 

This is align with equations numbers with one displayed equation: 
\begin{align}
x &= 4
\end{align}

This is align*: 
\begin{align*}
x &= 4 \\
& = 3+1.
\end{align*}






This is align: 
\begin{align}
x   &= 4 \\
    &= 3+1.
\end{align}

We can add comments: 
\begin{align}
x &= 4 
    & \textrm{by assumption} \\
& = 3+1.
\end{align}
